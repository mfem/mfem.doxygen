virtual void mfem::VisItDataCollection::SetPadDigits ( int  digits)
inlineoverridevirtual

Set the number of digits used for both the cycle and the MPI rank

Note
VisIt seems to require 6 pad digits for the MPI rank. Therefore, this function uses this default value. This behavior can be overridden by calling SetPadDigitsCycle() and SetPadDigitsRank() instead.
