Detailed Description

Logarithmic spacing function, uniform in log base 10 by default.

The log base can be changed as an input parameter. Decreasing it makes the distribution more uniform, whereas increasing it makes the spacing vary more. Another input option is a flag to make the distribution symmetric (default is non-symmetric). There are no scaled parameters. This function is nested.
